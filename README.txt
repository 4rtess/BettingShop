Чтобы запустить приложение нужно открыть командуню строку, перейти в каталог BettingShopParser_jar
После чего написать java -jar BettingShopParser.jar

!Важно!
Нужно установить chromedriver.exe вашей версии и заменить в папке BettingShopParser_jar
https://chromedriver.chromium.org/downloads

Чтобы узнать вашу версию хрома, в адресной строке хрома впишите chrome://version/
Первая строчка покажет версию.
